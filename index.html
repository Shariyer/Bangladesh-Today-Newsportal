Difference between arrow function and regular function

Ans: Arrow functions are only callable not constructible whereas, regular functions are both callable and constructible. Arrow function canot invoked with 'new' keyword on the other hand, we can use in regular function. when we use 'this' inside a regular function it refers the property of that object where it stays or inside the blockscope.But if we do the same inside an arrow function it refers window object or property and makes it global.
